# zhenxun-plugin-column
真寻机器人的竖版文字插件

## 文件说明
- `column.py`  
插件本体，放到`plugins`文件夹中即可。

## 使用说明
指令：`竖排<要转化成竖排的文字>`

## 注意事项
- 会将常用的中文标点符号对应替换成竖排样式的
- 会将半角空格替换成全角空格
- 不会将其他半角字符（英文字母、数字、英文标点符号等）转换成对应的全角
